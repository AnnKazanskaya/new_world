# Сайт-визитка ООО «Новый Мир»

Минималистичный техно-сайт для компании, занимающейся оптовой торговлей IT-оборудованием и системами охлаждения, в том числе с импортом из-за рубежа.

## Структура

- `index.html` — разметка одностраничного лендинга (hero, о компании, решения, логистика, контакты).
- `styles.css` — стили в тёмном техно-металик стиле с плавными анимациями.
- `script.js` — небольшие эффекты: анимации при скролле, актуальный год в подвале.
- `package.json` — dev-зависимость для удобного локального запуска.

## Запуск локально

1. Убедитесь, что установлен Node.js (LTS-версия).
2. В терминале перейдите в папку проекта:

```bash
cd "/Users/annakazanskaya/Desktop/Новый мир"
```

3. Установите зависимости:

```bash
npm install
```

4. Запустите локальный сервер:

```bash
npm start
```

После запуска откроется браузер со страницей `index.html`. Если не открылся автоматически — перейдите по адресу `http://127.0.0.1:4173`.

## Как менять контент

- Тексты блоков можно править прямо в `index.html` (заголовки, описания, пункты списков).
- Цвета, отступы и анимации настраиваются в `styles.css`.
- Логику форм, интеграцию с почтой или CRM можно добавить в `script.js` или подключить отдельный backend.
